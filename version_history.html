Version history of the SOG resource

Version 0.1: Announced to the pombe community on August 4, 2025, at the 12th International Fission Yeasts Meeting (Pombe2025) in Boston.

Version 0.2: Released on September 2, 2025. Changes from version 0.1 include:

  • Now uses a single SOG table as the entry point, and each SOG has its own SOG page.
  • Downloadable nucleotide sequences of CDSs and their 1-kb flanking regions are now provided on the SOG page.

Version 1.0: Released on December 30, 2025. Changes from version 0.2 include:

  • Enhanced OrthoGroup inference: Revised clustering workflow integrating OrthoFinder as the primary tool, with refinement using ProteinOrtho and PomBase-curated S. pombe ~ S. japonicus orthology to improve orthogroup accuracy and sensitivity.
  • Removed several poorly supported SOGs.
  • SOG ID continuity: IDs ≤ 5400 are retained from v0.2; new SOGs (IDs > 5400) were added in this release.
  • The SOG table is sorted by the genomic order of S. pombe genes within each OrthoGroup; SOGs lacking an S. pombe member are sorted in descending order by the number of genes within the OrthoGroup.
