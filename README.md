## ðŸ“– Overview
The **SOG** resource is a web platform designed to facilitate comparative genomics within the fission yeast genus. While *Schizosaccharomyces pombe* is a widely used model organism, the genus includes six other diverse species whose common ancestor dates back more than 200 million years.

This platform organizes genes from these species into over 5,000 orthogroups, allowing researchers to explore evolutionary relationships, gene conservation, and local synteny.

### Included species
1.  *Schizosaccharomyces pombe*
2.  *Schizosaccharomyces octosporus*
3.  *Schizosaccharomyces japonicus*
4.  *Schizosaccharomyces cryophilus*
5.  *Schizosaccharomyces osmophilus*
6.  *Schizosaccharomyces lindneri*
7.  *Schizosaccharomyces versatilis*

## âœ¨ Features
*   **Orthogroup search:** Query by systematic ID or *S. pombe* gene name.
*   **Chromosomal map:** Visualize the chromosomal locations of genes in an orthogroup.
*   **Phylogenetics:** Visualize phylogenetic trees for specific orthogroups.
*   **Sequence analysis:** View and download multiple sequence alignments (MSA).
*   **Synteny viewer:** Explore the conservation of local gene order (synteny) across species.
*   **Data retrieval:** Retrieve coding sequences (CDS) and flanking regions.

## ðŸ› ï¸ Technical implementation
This project is built as a **static web application**.
*   **Frontend:** HTML, CSS, and JavaScript.
*   **Hosting:** GitHub Pages via the custom domain `sogweb.org`.
*   **Backend:** None. The site relies on pre-computed data files served statically, ensuring high performance and zero maintenance.
